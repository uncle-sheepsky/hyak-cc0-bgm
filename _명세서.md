# cc0-bgm-pack — hyak CC0 음원 공개 패키지

hyak 채널 ep1~ep7 BGM의 **풀버전(2~3분 완결형)** 을 **CC0 1.0**(퍼블릭 도메인 헌정)로 공개하기 위한 배포 패키지. GitHub 릴리스 정본용 스테이징.

## 무엇
- 각 편 숏폼용 24~40초 루프와 별개로, **완결형 곡 구조**(인트로→빌드→코러스→브레이크→파이널→자연 엔딩)로 재편곡한 풀버전.
- **100% 코드 합성**(Python numpy/scipy, 외부 샘플·유료 라이브러리·VST 0). 빌더 `.py`가 곧 악보 → 결정론 재생성(SHA256 동일).

## 구성
```
cc0-bgm-pack/
  flac/   7트랙 무손실 (아카이벌)
  mp3/    7트랙 320kbps (편의)
  src/    풀버전 빌더 7개 + hyak_synth.py (신스 엔진)
  LICENSE       CC0 1.0 전문
  README.md     영문 배포 설명 + 트랙 매니페스트 + 재생성법
  MANIFEST.txt  길이 + SHA256 무결성
```

## 라이선스 근거
- ep1·3·4·5·6·7 = 완전 오리지널 코드합성.
- ep2 = 멜로디 코로베이니키(퍼블릭 도메인), 편곡·반주·드럼·합성 전부 오리지널. Tetris 상표 자산 미사용.
- ep7 = 100% 코드합성 오리지널(제작자 본인 duru 원곡 편곡). **東方 등 제3자 소재 0** → 원작 무관, CC0 유효. (구 라이선스 감사 §6의 "ep7 CC0 자발 제외"는 BGM이 두루 오리지널로 확정됨에 따라 해제.)
- 권리자 = hyak 단독. CC0 헌정 = **게시(공개 배포) 시점에 성립** — 사용자 게이트.

## 배포 워크플로 (미완 = 사용자 게이트)
1. GitHub 리포 생성 → 이 폴더 커밋 → **릴리스**로 flac/mp3 zip 첨부(정본).
2. 유튜브 커뮤니티 탭에 릴리스 링크 게시(문안 = `community-post.md`).
3. (선택) archive.org 미러 업로드 — 비개발자 발견성.

## 재생성
`cd src && python build_track_ep3_fullver.py` (numpy·scipy 필요). ep7만 duru_synth 추가 의존(스탠드얼론 미실행, 렌더본 동봉).

## 소스 위치
빌더 원본: `영상소스/음원/build_track_ep1_fullver.py`, 각 `영상소스/ep*/build_track_ep*_fullver.py`. 원본 wav도 각 소스 폴더 잔존(이 패키지는 flac/mp3만 배포).
